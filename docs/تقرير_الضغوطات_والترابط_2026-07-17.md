## أولاً: ورشة ترابط البيانات (data-integrity-forge)

خريطة العلاقات (من الكود): كتالوج الملفات → مجموعات المحتوى؛ لقطات النسخ → الكتالوج؛
متجهات (Mongo/قرص) → المجموعات؛ الجلسات → الحسابات؛ مشتق: latest_version = أعلى لقطة.

| العمود | قبل | بعد الصهر |
|---|---|---|

| كاسكيد كامل | ✅ (مُثبت **تفجيرياً**: نشر ملف مؤقت ثم حذفه → صفر بقايا في المجموعة والكتالوج واللقطات ومتجهات القرص وMongo) | ✅ |

**إثبات إضافي**: حارس جلسات الزوار — سؤال زائر حي لم يُنشئ أي جلسة (8→8).
**الأدوات دائمة** في `أدوات/`: `سبر_الترابط.py` (قراءة فقط — شغّله متى شئت) و`سبر_تفجيري.py` (دورة كاملة على الخادم المحلي، ينظف نفسه حتى عند الفشل).

## ثانياً: جولة التجربة الذاتية (study-pressure)

| الميزة | شو جرّبت | النتيجة |
|---|---|---|
| صفحة الدخول | دخول خاطئ → رسالة عامة بلا كشف السبب، بلا تخزين جلسة | ✅ |
| الدخول الصحيح | طالب تجربة مزروع → تحويل للشات باسمه ورقمه | ✅ |
| الشات + البث + البيانات الشخصية | «حالتي الأكاديمية ومعدلي؟» → «ممتاز، 88.5%» من ملفه المزروع بالضبط | ✅ |
| الخروج | مسح الجلسة والعودة للبوابة | ✅ |
| كونسول المتصفح | طوال الجولة | ✅ صفر أخطاء |
| مدخلات مؤذية (9) | سؤال فارغ/6000 حرف/7 أدوار/دور 9000 حرف/JSON مكسور → 422، بلا توكن → 401، ملف وهمي → 404 موحدة، دخول خاطئ → 401 عامة | ✅ 9/9 |
| 10 أسئلة متنوعة | أرقام/قوائم/كيانات/عامية/خارج نطاق/تجميعي/متابعة ذاكرة | ✅ 10/10 (أبرزها: «وهل الطب منها؟» بعد سؤال معدل 90 → رفض صحيح بمرجعية 91% لعام 2025-26) |

**عطل بيئي أثناء الشغل**: انقطاع إنترنت قطع الدورة التفجيرية الأولى في منتصفها —
نُظفت البقايا (حساب مؤقت + مسودة + لقطة) وأعيدت الدورة حتى الاخضرار الكامل.
وخطآن في سكربتَي الفحص نفسيهما (نمط اسم ملفات القرص المجزأ + مجلد العمل) صُحّحا —
النظام كان سليماً والفحص هو الذي كان خاطئاً، وسُجل ذلك بصدق.

## ما لم يُجرَّب ولماذا

- لوحة الأدمن والموظف بالواجهة الرسومية كاملة (زر بزر) — جُرّبت مساراتها عبر الـ API
  (رفع/نشر/حذف/صلاحيات) في هذه الجلسة وسابقاتها، لكن جولة أزرار شاملة لواجهتيهما
  لم تحصل اليوم.
- حدود المعدل بالفيضان (429) — تفادياً لحرق حصة مفتاح التجربة.
- البث على Render — ممنوع لمس مفتاح Render في التجارب بقرارك.
